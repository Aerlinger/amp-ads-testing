by Vamsee Jasti jastiv@

The Google Times

This is an AMP page which has delivered an ad via DFP. On click, AMP will eliminate the redirect to the adserver and instead send the user directly to the landing page.
Moreover, since the landing page is in AMP, the landing page will load in an instant.
AMP ad landing pages work in both cases where hosting page is AMP or non-AMP.
In order to make AMP landing pages work, include this tiny peice of JavaScript in your creative code:

<script async src="https://cdn.ampproject.org/alp.js"></script>
